Karnik Time

Update 6/30/2022

What’s New

Gusto Payroll Import

  • Added support for the Gusto payroll import format.
  • Specifications followed are located on Gusto’s support website.

Weekly Pay Period

  • Added support for “Weekly” pay period
  • Weekly pay period runs from Sunday through Saturday.
  • Bi-Weekly pay periods are still supported.
  • When switching between pay periods salaries will be automatically adjusted appropriately.

Installation

  1. Download zip file from https://hoipkemier.com/KarnikTime/KarnikTime_Update_20220630.zip
  2. Unblock the zip security.
  1. Right click on the zip file, and select properties.
  2. Check the “Unblock” checkbox.
  3. Click Ok to save changes.
  1. Extract files to a temporary location.
  2. Locate the directory of the current executable.
  1. Right click on the Karnik Time Shortcut.
  2. Click on properties.
  3. Take note of the Start In directory.
  1. Copy the files from the zip to the “Start in” directory.
  1. Overwrite any files when prompted.

NOTE:  Do not delete all the files in the target directory.  There are several configuration files that will not be replaced, and will remain from the previous installation.
